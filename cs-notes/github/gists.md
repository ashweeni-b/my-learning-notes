### Gists

- Feature of GitHub that allows users to share code snippets, notes, or other small pieces of information in a lightweight and convenient way
- Mini Git repo - fork, clone, version control
- Useful for sharing quick solutions, config files or examples

#### Key Features of Gists

1. Public Gists - Visible to everyone and searchable. Publicly listed <br />
    Secret Gists - Not private and searchable. Anyone with the link can access

1. Version Control

1. Forking and Cloning

1. Embedding <br />
    Gists can be embedded into websites or blogs <br />
    Used for sharing code examples in tutorial or documentation

1. Markdown Support

1. Collaboration

#### Use Cases for Gists

1. Sharing quick code examples or solutions
1. Sharing configuration files or scripts for personal use
1. Creating templates for commonly used code patterns
1. Sharing error logs or debugging information with others
1. Embedding code snippets in blogs, forums, or documentation

#### Limitation of Gists

1. Not used for sharing sensitive or confidential information
1. Best suited for small snippets and single files not for larger projects or multi-file structure
