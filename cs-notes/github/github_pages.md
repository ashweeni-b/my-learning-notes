### GitHub Pages

- Free static sites hosted directly from GitHub repository
- Use of Markdown or YAML files are there for GitHub Pages
- Site generation technologies like Jekyll and Liquid are used to define dynamic templates into static templates

#### Enabling GitHub Pages

1. Navigate to the repo page
1. Select _Settings_
1. Scroll down to GitHub Pages
1. Use the main branch or the docs folder from the repo
1. Select the theme from drop-down (e.g. Jekyll)
