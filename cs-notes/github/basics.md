#### How to create a repository in GitHub?

1. In the upper-right corner of any page, use the drop-down menu, and select _New Repository_
1. Use the owner drop-down menu to select the account in which the repo needs to be created
1. Type a name for the repo and an optional description
1. Choose repo visiblity - Public or Private
1. Select _Create Repository_

#### How to add a file to the repository in GitHub?

1. Navigate to the main page of the repo
1. In the repo, browse to the folder where the file needs to be created
1. Click on _Add file_ drop-down menu in the upper-right corner and select _Create new file_
1. In the file name, type the file name along with the extension of the file. For creating directories, type the '\/'seperator (directory separator)
1. In file content text box, type content of the file
1. To review the file content select _Preview_ from above the file contents
1. Select _Commit changes_
1. Type a short and meaningful commit message in the commit message field
1. Select the appropriate email address from the drop-down menu
1. Decide the branch on to which the commit needs to be added
1. Select _Commit changes_ or _Propose changes_
