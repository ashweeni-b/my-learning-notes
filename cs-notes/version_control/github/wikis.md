### Wikis

- Section for hosting documentation
- Used to share long-form content about the project, such as how to use it, how its designed or its core principles
- README quickly tells what the project does but Wikis can be used for sharing additional information

#### Creating Wiki pages using GitHub

1. Navigate to the repository
1. Select the _Wiki_ tab
1. Select _Create the first page_, if no page exists, or _New Page_ to add a new page
1. Enter a title and content, then select _Save Page_

#### Editing Wiki pages using GitHub

1. Navigate to the Wiki page 
1. Select _Edit_ at the top-right corner
1. Make changes and select _Save Page_

#### Deleting Wiki pages using GitHub

Deleting a Wiki page requires Git

1. Navigate to the project repo
1. Clone the project repo
1. Remove the file
1. Push the changes
