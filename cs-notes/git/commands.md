### Commands in Git

#### Git Bash Commands

1. ```git --version``` - Tells the version of Git installed in the system
    
1. ```cd``` - Tells the current directory

1. ```cd <folder-name>``` - Changes the current directory to the _folder-name_

1. ```cd ..``` - Moves one directory down in the hierarchy

1. ```cd ../..``` - Moves two directory down in the hierarchy

1. ```cd /~``` - Moves to the initial directory where Git is installed

1. ```mkdir <folder-name>``` - Create folders in the directory

1. ```touch <file-name>``` - Create files in the directory

1. ```mv <old-name> <new-name>``` - Rename the folder or file in the directory

1. ```mv <file-name> <destination-folder-name>/``` - Moves file from one directory to another directory

1. ```ls``` - Lists all the files and folders from the directory

1. ```cp <file-name> <folder-name>``` - Copies the files into the folder

1. ```rm <file-name>``` - Deletes the file

1. ```rm -rf <folder-name>``` - Deletes the folder

> <i> -r: Recursive - Allows the folder deletion along with its subfolders and files. Deletes the sub-directories also. <br />
-f: Force - Deletes directory without prompting for confirmation. </i>

1. ```clear``` - Clears the Git Bash screen
