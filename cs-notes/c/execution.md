#### Steps for Execution of Program

1. Source program is written in C language using VS Code as code editor.
1. The source program is then given to the Preprocessor.
1. Header files containing directives and prototypes are combined with the source file.
1. The source file is then further given to the Compiler.
1. Compiler converts the high-level programming language into low-level machine code.
1. An object file is created after compilation.
1. The object file is then given to the Linker.
1. The compiled program is then linked with library routines and this process also combines all intermediate object files.
1. An executable file_(.exe)_ is generated which when executed gives the output of the program.

#### Basic Execution of C Program

1. Every program's execution starts with the _main()_ function.
1. All the program statements are terminated with a semicolon.
1. Program instructions are case-sensitive.
1. Instructions are executed in the same order in which they are written.
