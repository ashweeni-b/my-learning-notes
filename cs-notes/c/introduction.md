### C Programming Language

- Mother of all programming languages
- Developed by _Dennis Ritchie_ in 1972 at AT \& T Bell Labs
- One of the first stable programming language
- Foundation of C\+\+, Java and Python
- Developed for writing the Unix operating system

#### Why C\?

Initially developed for Unix OS, but in current time it is used for the development of Embedded System, Real Time Algorithms and Operating System

- A Language - Martin Richard in 1960s
- B Language / BPCL - Ken Thompson in 1969 
- C Language - Dennis Ritchie in 1972

#### Uses of C

1. Major parts of Windows, Linux and other OS are built in C
1. To write driver programs for devices like tablets, printers, etc.
1. To program embedded systems where program needs to run faster in limited memory, e.g. Microwave, Cameras, etc.
1. Develop games where latency is very important
